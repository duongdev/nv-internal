RESPONSIVE BEHAVIOR
===================

iPhone SE (Small Screen):
- Dialog takes up more vertical space
- Buttons stack vertically on small screens
- Font sizes remain readable

iPhone Pro Max (Large Screen):
- Dialog centered with padding on sides
- Buttons remain horizontal
- More whitespace around content

iPad:
- Dialog max width: 500px
- Centered in screen
- Same behavior as iPhone


DARK MODE
=========

All components support dark mode:
- Background: Uses theme background color
- Text: Uses theme foreground color
- Buttons: Uses theme destructive color for delete actions
- Dialogs: Uses theme card background
